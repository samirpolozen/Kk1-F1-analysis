# Formel 1 Analys
I denna notbook analyseras det hur startposition påverkar resultatet i Formel 1.

Analysen använder sig av "results.csv" från Kaggle och siktar in sig på dessa tre påstående:

1. Bättre startpositioner ger fler poäng i genomsnitt genom historien
2. Bättre startposition ger större chans till första plats
3. Startpositionen har stor betydelse

Dataset: https://www.kaggle.com/datasets/rohanrao/formula-1-world-championship-1950-2020
